Financial Market Volatility Forecasting

This repository contains reproducible materials for my Postdoctoral Fellowship project conducted at Masaryk University.

Project Title
Advanced Fuzzy and Hybrid Models for Financial Market Volatility Forecasting

Contents
- scripts/: R codes for models
- synthetic_data.csv: demo dataset
- results_tables.xlsx: benchmark results
- session_info.txt: software versions
- data_access.txt: instructions for obtaining original data

1. Fuzzy EWMA
2. RV-FTS (Realized Volatility Fuzzy Time Series)
3. Benchmarks:
   - HAR
   - GARCH
